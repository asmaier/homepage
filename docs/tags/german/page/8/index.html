Full-HD Auflösung unnötig!

Drei interessante Zitate aus einem Testbericht der ct zum Thema Full-HD LCD-Fernseher: Die theoretische Auflösung des normalsichtigen Auges beträgt 1 Winkelminute oder umgerechnet auf zwei Meter Betrachtungsabstand „1 Fliegenschiss“ (0,6 mm).
