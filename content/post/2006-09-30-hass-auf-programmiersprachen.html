  • C-Hasser in 10 Tagen
Leider muß ich zugeben, dass er mit vielem Recht hat. Er hasst auch Skriptsprachen:
  • Skriptsprachen-Hassen leichtgemacht
Und wieder hat er mit vielem Recht. Aber eine Sprache zu hassen hat er vergessen: FORTRAN! Selbst das modernere FORTRAN90 hat viele Überaschungen auf Lager, die man in anderen Programmiersprachen "vermisst":
  • Mistakes in Fortran 90 Programs That Might Surprise You
Auch wenn man unverständlichen Code schreiben möchte, ist FORTRAN einfach die beste Wahl
Write all your code in FORTRAN. If your boss ask why, you can reply that there are lots of very useful libraries that you can use thus saving time. However the chances of writing maintainable code in FORTRAN are zero, and therefore following the unmaintainable coding guidelines is a lot easier.
(aus How To Write Unmaintainable Code)
FORTRAN steht praktisch als Synonym für schlechte Programmierweise
“You can write Fortran in any language.”
(aus How Not to Write FORTRAN in Any Language)
Aber echte Programmierer lassen sich davon natürlich nicht abschrecken:
  • Real programmers don't use Pascal!
Echte Programmierer benutzen FORTRAN gemischt mit C!
